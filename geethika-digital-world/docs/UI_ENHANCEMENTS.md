## Design System

### Colors
- Valentine Red: #DC143C
- Valentine Pink: #FF69B4
- Valentine Rose: #FF007F
- Valentine Light Pink: #FFB6C1
- Valentine Dark Red: #8B0000

### Typography
- Display Font: Playfair Display (serif, italic for emphasis)
- Body Font: Inter (clean, modern)

### Animations
- Float: 6s ease-in-out infinite
- Fade In: 1s ease-out
- Slide Up: 0.8s ease-out
- Staggered delays for grid items

### Spacing
- Section padding: 16-20 (py-16 md:py-20)
- Container: max-w-7xl with responsive padding
- Grid gaps: 4-6 (gap-4 md:gap-6)
